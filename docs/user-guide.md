# CrisisLens User Guide

Welcome to CrisisLens! This guide will help you get started with using the platform.

## Overview

CrisisLens is a real-time crisis intelligence platform that helps you:
- Monitor crisis events from multiple sources
- Verify claims and misinformation
- Track crisis evolution in real-time
- Collaborate with teams
- Access data via web, mobile, and API

## Getting Started

### 1. Create an Account

Visit [https://app.crisislens.io](https://app.crisislens.io) and click "Sign Up".

### 2. Dashboard Overview

The dashboard shows:
- **Active Crises**: Currently monitored events
- **Pending Review**: Items awaiting verification
- **Risk Heat Map**: Geographic view of risk levels
- **Recent Updates**: Live feed of new items

### 3. Navigation

- **Home**: Dashboard and overview
- **Search**: Find specific events or topics
- **Map**: Geospatial visualization
- **Claims**: Review fact-checking queue
- **Analytics**: Statistics and insights
- **Profile**: Settings and preferences

## Core Features

### Monitoring Crisis Events

**View All Items:**
1. Click "Items" in navigation
2. Use filters (status, risk level, location)
3. Click any item for details

**Item Details Include:**
- Title and full content
- Risk score (0-100%)
- Source information
- Location and map
- Related claims
- Evidence and verification

### Claim Verification

**Verify a Claim:**
1. Go to "Claims" section
2. Select pending claim
3. Review evidence
4. Choose verdict:
   - ✅ True
   - ❌ False
   - ❔ Uncertain
5. Add supporting evidence
6. Submit verification

### Search and Filters

**Search Options:**
- **Keyword search**: "earthquake japan"
- **Location**: Filter by region
- **Time range**: Last hour, day, week
- **Risk level**: High, medium, low
- **Status**: Pending, verified, rejected

**Advanced Filters:**
```
risk_score >= 0.7 AND location:Tokyo AND status:pending
```

### Real-time Updates

Items update automatically when:
- New events are detected
- Verification status changes
- Risk scores are recalculated
- Team members add notes

**Enable Notifications:**
1. Go to Profile > Notifications
2. Choose channels (email, push, SMS)
3. Set alert thresholds
4. Save preferences

### Map View

**Interactive Map Features:**
- Color-coded risk levels
- Cluster view for multiple events
- Click markers for details
- Filter by region
- Heat map overlay

### Collaboration

**Team Features:**
- Assign items to team members
- Add internal notes
- Track review history
- Shared workspaces

**Commenting:**
1. Open item details
2. Scroll to comments
3. Add your comment
4. @mention team members
5. Submit

## Troubleshooting

### Common Issues

**Can't see new items?**
- Refresh the page
- Check your filters
- Verify network connection

**Notifications not working?**
- Check browser permissions
- Verify email settings
- Test notification settings

**Map not loading?**
- Clear browser cache
- Check internet connection
- Try different browser
